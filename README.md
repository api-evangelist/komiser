# Komiser (komiser)
Komiser is an open-source, cloud-agnostic resource manager for analyzing and managing cloud cost, usage, security, and governance across multi-cloud environments including AWS, Azure, GCP, DigitalOcean, OCI, Linode, Scaleway, Tencent, MongoDB Atlas, and Kubernetes.
